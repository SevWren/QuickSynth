status: active
owner: QuickSynth maintainers
---

Analyze the text internally (don't show questions). Create a concise summary:
- Use simple sentences (split complex ones).
- Use bullets for key facts/details.
- Bold the first 2-3 words of new ideas.
- Retain important clarifying details (scope, caveats).
- If long, use 2-3 paragraphs.
- Formatting: **Bold** central theme & key facts/entities. *Italicize* supporting ideas, author's purpose, & implications/outcomes.
- Output ONLY the formatted summary.
